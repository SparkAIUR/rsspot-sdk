`rsspot` is an async-first Python SDK and CLI for Rackspace Spot. Use it to automate organization, cloudspace, inventory, pricing, and nodepool workflows with typed models and profile-aware auth.

## Capability Matrix

| Capability | What you get | Primary entrypoint |
|---|---|---|
| Local profile management | Multi-account config with active profile switching | `rsspot configure`, `rsspot profiles *` |
| API automation via CLI | Direct task execution with json/yaml/table output modes | `rsspot <group> <command>` |
| API automation via SDK | Async client with typed Pydantic response models | `SpotClient`, `get_client` |
| Auth lifecycle | Lazy token refresh with retry-aware request flow | `SpotClient.authenticate()` |
| Schema drift tracking | OpenAPI snapshot + generated operation index | `scripts/sync_openapi.py`, `scripts/generate_openapi_index.py` |
| Docs publishing | Sparkify + GitHub Pages workflow | `.github/workflows/docs-pages.yml` |

## Quickstart

```bash
uv sync
uv run rsspot configure \
  --profile default \
  --org <org-name> \
  --region us-central-dfw-1 \
  --refresh-token "$SPOT_REFRESH_TOKEN"
uv run rsspot organizations list --output table
```
